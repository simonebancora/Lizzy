Roadmap
=======


Lizzy is an evolving project, and new features are continuously being developed and added. Below is a roadmap of planned features:

**Short term:**

- Additional boundary condition types (flow rate, you're next).
- Support for additional mesh import formats.
- Support for 3D elements.

**Long term:**

- Lizzy becomes the world best solver for resin infusion simulations.

Timeline
--------

- 2021: A primordial FE/CV Matlab solver is written as a passion project.
- 2023: Codebase is ported to Python.
- 2025: Lizzy becomes public with v0.1.0 and real development starts.
- 2026: Lizzy v1.0.0 is released.
